surly
=====

Node.js AIML interpretter


Requirements
============

 * [Node.js](http://nodejs.org/)
 * [NPM](https://www.npmjs.org/)
 * [Grunt](http://gruntjs.com/)


Installation
============

 1. Clone the repo
 2. `$ npm install`
 3. `$ bower install`
 4. `$ grunt`


Usage
=====

 1. `$ sudo node server.js`
 2. Browse to http://localhost (or whatever)
